    1. What is Semantic HTML? 
        Semantic HTML has tags with meaning.
    2. What is HTML used for? 
        HTML is used for displaying data in a graphical form. It's the skeleton of a website.
    3. What is an attribute and where do we put it? Attributes are placed in opening tags and are used to include information that will not be rendered on the webpage.
    4. What is the h1 tag used for? 
        How many times should I use it on a page? <h1> is used for the most important heading. It should only be used once.
    5. Name two tags that have required attributes
        <a href=" "></a> , <img src=" " />
    6. What do we put in the head of our HTML document? 
        CSS links, metadata
    7. What is an id? 
        An id is an attribute that can be used as a CSS selector to select a specific element.
        An id can only be assigned to one element.
    8. What elements can I add an id to? 
        Any element.
    9. How many times can I use the same id on a page? 
        Once.
    10. What is a class? 
            A class is an attribute that can be assigned to multiple elements. Used as a CSS selector to select specific element. 
    11. What elements can I add a class to? 
        Any element.
    12. How many times can I use the same class on a page? 
        As many as you want!
    13. How do I get my link to open in a new tab?
        Put target="_blank" in opening tag.
    14. What is the alt attribute in the image tag used for? 
        Describes the picture for accessibility/in case the image doesn't show up.
    15. How do I reference an id?
        #
    16. What is the difference between a section and a div?
        A section is more specific than a div. Sections are semantic, divs are not.
    17. What is CSS used for? 
        CSS is used to style or decorate a webpage.
    18. How to we select an element? Example - every h2 on the page
        h2{}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
        A class can be used on multiple different elements, but an id can only be assigned to one element.
        You might use a class when you need to format more than one part of a webpage the same way. 
        You would use an id for an element that only appears on the page once.
    20. How do we select classes in CSS?
        .class{}
    21. How do we select a p element with a single class of “human””?
        .human p{}
    22. What is a parent child selector? When would this be useful? 
        This selects an element (child) that is nested inside another element (parent).
        This is useful if you need to select all paragraphs within a div. 
    23. How do you select all links within a div with the class of sidebar?
        .sidebar a:link{}
    24. What is a pseudo selector?
        Used to define a special state of an element.
    25. What do we use the change the spacing between lines?
        line-height
    26. What do we use to change the spacing between letters?
        letter-spacing
    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
        text-transform: uppercase; text-transform: lowercase; text-transform: capitalize;
    28. How do I add a 1px border around my div that is dotted and black?
        div{border: 1 px dotted black}
    29. How do I select everything on the page? 
        *{}
    30. How do I write a comment in CSS?
        /*   */
    31. How do I find out what file I am in, when I am using the command line?
        pwd 
    32. Using the command line - how do I see a list of files/folders in my current folder?
        ls
    33. How do I remove a file via the command line? Why do I have to be careful with this? 
        rm -r
        You won't be prompted before deleting and the files will be gone forever.
    34. Why should I use version control? 
        Version control allows several developers to add content at the same time and keeps an updated history of changes to the content.
    35. How often should I commit to github?
        every 20 minutes
    36. What is the command we would use to push our repo up to github?
        git push -u origin firstName-lastNAme 
    37. Walk me through Lambda's git flow. 
        1. Fork repo to make your own copy. Ensure that your github username is in the top left. 
        2. Add collaborators: settings>Collaborators>type PM name>Add collaborators
        3. Copy and paste clone URL. git clone <url>
        4. Make a branch on your fork. 'firstName-lastName". git branch
        5. Submit pull request. Use your fork as base-fork. Add PM as reviewer. 
        6. As a project manager, accept the merge request after conducting review of the code.

        inline elements such as span do not start a new line, but block elements such as div do.
    2. What happens when an element is positioned absolutely? 
        it is positioned relative to the nearest positioned ancestor.
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
        display: inline-block

        display block: <div>, <section>, <p>. display inline: <span>, <a>. display inline-block: <button>
